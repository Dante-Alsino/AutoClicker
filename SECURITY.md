# Política de Segurança

## Versões Suportadas

Apenas a versão mais recente (stable) recebe atualizações de segurança.

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Se você descobrir uma falha de segurança, por favor **NÃO** abra uma Issue pública imediatamente.

Envie um e-mail para o mantenedor (ou entre em contato via GitHub privado) com os detalhes. Nós faremos o possível para corrigir o problema rapidamente.

Obrigado por ajudar a manter o projeto seguro!
